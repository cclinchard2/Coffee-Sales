Coffee Sales Reports

Explore the best predictors of coffee sales revenue using Python, R, and Tableau.

This project examined different impacts on coffee revenue.

Tableau was used for visualizations, and the analyses (stepwise regression and multiple linear regression) were done in both R and Python (replicated).

PDFs from R and Python are available to show the code used, and the coffee sales report provides an overview of the findings.
